├─────────────────────────────────────────────────────────┤
│ Toolbar (48px)                                        │
│ ├─ View Switcher ├─ Date Navigator ├─ [折叠全部][展开全部] ├─ Actions │
├─────────────────────────────────────────────────────────┤
│                                                         │
│ Main Content                                           │
│ ┌─────────────┬────────────────────────────────────┐   │
│ │             │                                    │   │
│ │ Sidebar     │    Gantt Chart Area              │   │

│                                                         │

**详细设计：**

**右上角用户区（User/Sync）**

- 未登录：按钮“登录”，次要链接“继续本地模式”
- 已登录：头像/邮箱 + 下拉菜单（账号与同步、退出登录）
- 同步状态徽标：
  - 已同步：绿色点
  - 同步中：转圈图标
  - 离线待同步：灰色“离线”标签 + 待同步数量
  - 冲突：红色“⚠ 冲突”标签（可点击进入处理）

**Toolbar 操作区**

- 视图切换（日/周/月/年）
- 日期导航：
  - `‹` / `›` 箭头按钮：逐年前进/后退
  - **年份数字**：可点击，点击后在数字下方弹出年份下拉选择器（dropdown），列表可滚动，范围为当前年份前后各10年；选中某年后立即切换并关闭下拉；点击下拉外区域关闭；样式与 ghost-btn 风格一致，最大高度240px，超出时内部滚动
- **折叠/展开全部**：两个并排的 ghost 按钮"折叠全部"/"展开全部"；点击折叠全部收起所有管线下的需求行，点击展开全部恢复显示所有需求及环节行；按钮始终显示在 Toolbar 中（不隐藏到菜单）；**初始状态为展开全部**
- **导出 Excel**：ghost 按钮"↓ 导出Excel"；导出当前视图的甘特图，受筛选影响；点击直接下载 `.xlsx` 文件；**Excel 单元格格式**：
  - 表头行（时间刻度）及管线行：Bold + 水平居中，字号 22pt
  - 需求行：Bold + 水平居中，字号 20pt；行上方加粗边框（2pt 实线，与管线/需求块区分）
  - 环节行：不加粗 + 水平居中，字号 18pt
- **导入 Excel**：ghost 按钮"↑ 导入Excel"；点击弹出文件选择框（仅接受 `.xlsx`/`.xls`）；系统自动识别两种格式：
  - **固定列格式**（通用迁移格式）：A=管线名称、B=需求名称、C=环节名称、D=开始日期（条件）、E=结束日期（条件）、F=需求级别、G=里程碑、H=日期区间（条件，`YYYY-MM-DD - YYYY-MM-DD`，D/E 均为空时使用）
  - **甘特图矩阵格式**（本网站导出文件）：第一行为时间刻度列头（自动检测）、第一列为管线/需求/环节层级、有颜色填充的单元格即为环节排期
  - 上传后显示导入结果面板：成功需求数、失败行号与原因、自动新增的管线数与环节库条目数
- **下载导入模板**：ghost 按钮"↓ 下载导入模板"；点击直接下载固定列格式 `.xlsx` 模板文件（含 8 列表头：管线名称/需求名称/环节名称/开始日期/结束日期/需求级别/里程碑/日期区间，含示例行与列说明）；矩阵格式无需单独模板，直接使用本站导出文件导入

**时间轴头部 (Time Header)**

- 高度：40px
- 背景：#F5F3FF
- 显示：视图相关的时间刻度（见下）
- 支持横向滚动，显示范围根据视图变化
- 当前日期用淡紫色竖线高亮显示

**时间刻度文案（Header Label）**

- 日视图：`MM-DD`（示例：`05-20`）
- 周视图：`MM-DD`（示例：`05-18`，表示该周起始日）
- 月视图：`YYYY-MM`（示例：`2026-05`）
- 年视图：`yyyy`（示例：`2026`）

**筛选控件（环节/级别/管线）**

- 形态：圆角胶囊/平滑圆角（避免棱角长方形观感）
- 边框：1px `#DDD6FE`
- 交互：hover/focus 时边框与背景色平滑过渡（120ms\~150ms）

**时间网格线（Grid Lines）**

- 日/周/月刻度必须用纵向线分隔，贯穿Header与甘特画布
- 网格线颜色要求：比左侧Sidebar分隔线更淡（避免抢占视觉焦点）

**左侧Sidebar（树状图）**

- 左侧为树状结构：管线 → 需求 → 环节
- 左侧宽度支持拖拽调节，最小宽度：60px

**任务条 (Task Bar)**

- 高度：24px
- 圆角：4px
- 颜色优先级：环节库自定义色 → 管线颜色
- 显示：任务名称、进度百分比
- 拖拽手柄：左右边缘3px可拖拽区域
- **Hover Tooltip**：鼠标悬停时在鼠标附近弹出浮层，显示：
  - 环节名称（加粗）
  - 日期区间：`YYYY-MM-DD ~ YYYY-MM-DD`（单行，起止日期以 `~` 连接）
  - 浮层背景 `#1f2937`（深色），文字白色，圆角 6px，内边距 8px 12px，`pointer-events: none` 不干扰拖拽
  - 拖拽进行时不显示 Hover Tooltip（改为显示拖拽实时 Tooltip）

- **拖拽实时 Tooltip**：拖拽进行中，在鼠标附近实时显示变更后的日期（与 Hover Tooltip 同款深色浮层风格，跟随鼠标更新，松手后消失）：
  - 拖动中间（平移）：`YYYY-MM-DD ~ YYYY-MM-DD`（新起止日期区间）
  - 拖动左边缘（调整开始日期）：`开始：YYYY-MM-DD`
  - 拖动右边缘（调整结束日期）：`结束：YYYY-MM-DD`

**依赖线 (Dependency Line)**

- 样式：虚线，1px
- 颜色：#94A3B8
- 箭头：指向下一个任务
- 动画：拖拽时实时更新

### 2.2 开发范式页面

**布局结构：**

```
┌─────────────────────────────────────────────────────────┐
│ Header (同首页)                                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│ Main Content                                           │
│ ┌────────────────────────┬────────────────────────────┐ │
│ │ Paradigm List (左侧)   │  Stage Editor (右侧)       │ │
│ │                        │                            │ │
│ │ ├─ Category A          │  ├─ Stage Name             │ │
│ │   ├─ Paradigm 1        │  ├─ Mandays                │ │
│ │   └─ Paradigm 2        │  ├─ isMilestone            │ │
│ │ ├─ Category B          │  ├─ Dependencies (多条)    │ │
│ │   └─ Paradigm 3        │  └─ Actions                │ │
│ │                        │                            │ │
│ │ [+ 新建模板]           │  [+ 添加环节]              │ │
│ │ [↓ 下载Excel模板]      │                            │ │
│ │ [↑ 批量导入Excel]      │                            │ │
│ └────────────────────────┴────────────────────────────┘ │
└─────────────────────────────────────────────────────────┘
```

**详细设计：**

**范式列表**

- 宽度：40% 或 400px
- **高度固定，内部三段式布局**：左侧卡片高度固定为视口可用高度（`height: calc(100vh - header - padding)`），不随右侧编辑区内容伸长；内部分为三个区域：
  1. **顶部**：模板分类管理（固定，不滚动）
  2. **中部**：范式模板列表（`flex: 1; overflow-y: auto`，内容超出时独立滚动）
  3. **底部**：下载Excel模板 / 批量导入Excel 两个按钮（固定在卡片底部，始终可见，不随列表内容上移）
- 分组显示：按分类折叠/展开
- 选中状态：左侧4px淡紫色边框
- 操作按钮：悬浮显示删除按钮
- 列表底部固定显示两个操作入口：
  - **下载Excel模板**：下载预填好表头和示例行的 `.xlsx` 文件
  - **批量导入Excel**：上传填写完毕的模板文件，触发批量创建范式

**阶段编辑器**

- 卡片式布局，每个阶段一个卡片
- 拖拽排序：支持拖拽调整顺序
- **范式基本信息区**（编辑器顶部，在环节列表上方）：
  - 范式名称（文本输入）
  - 模板分类（下拉）
  - **所属管线**（下拉，列出系统已有管线，首项为「不限」/空）；选择后影响下方环节 combobox 的优先排序
- **环节名称字段：combobox 风格（可输入可选择）**
  - 范式**已选所属管线**时：下拉列表顶部显示「本管线环节」分组（该管线下的环节库条目），下方显示「其余环节」分组；两组均支持实时搜索过滤
  - 范式**未选所属管线**时：展示全部环节库条目，不分组
  - 选中后自动预填 `stageCategory`（类别）字段，用户仍可修改
  - 环节库为空时退化为纯文本输入
- 依赖选择：多条依赖规则列表，每条可独立配置前置环节、关系类型、触发方式、偏移值
- 实时保存：输入后2秒自动保存

**Excel 模板格式**

模板列顺序与飞书/Excel 甘特图常见格式对齐，便于直接复制粘贴已有排期数据：

| 列  | 列头         | 示例值           | 说明                                                                                                                |
| --- | ------------ | ---------------- | ------------------------------------------------------------------------------------------------------------------- |
| A   | 范式名称     | 通用生产项目范式 | 同一范式的所有环节行填写相同值，系统据此分组                                                                        |
| B   | 环节名称     | 需求设计         | 必填，同一范式内不可重复；**若环节库有条目则提供 Excel 原生下拉选项，也可直接填写新名称（导入后自动追加至环节库）** |
| C   | 参考人天     | 3                | 正整数                                                                                                              |
| D   | 前置依赖环节 | 开发实现         | 填写同范式内B列的环节名称；多个用英文逗号分隔                                                                       |
| E   | 依赖偏移天数 | 1                | 整数；仅在触发方式为offset时生效，留空默认0                                                                         |
| F   | 依赖关系类型 | FS               | 下拉：`FS`（完成后开始）/ `SS`（开始后开始），留空默认FS                                                            |
| G   | 触发方式     | finish_100       | 下拉：`finish_100` / `finish_percent` / `finish_offset_days` / `start_offset_days`，留空默认finish_100              |
| H   | 里程碑节点   | L1               | 下拉：`L0` / `L0.5` / `L1` / `L2`；留空表示非里程碑节点                                                             |
| I   | 模板分类     | 通用             | 下拉：对应系统已有分类，留空归入"通用"                                                                              |

**批量导入交互**

- 点击"批量导入Excel"弹出文件选择框，仅接受 `.xlsx` / `.xls`
- 上传后系统预校验，展示导入结果面板：
  - 成功：显示创建的范式数量，列表即时刷新
  - 部分失败：成功范式正常导入；失败范式以表格形式展示首个错误行号、范式名称、错误原因
  - 全部失败：不导入任何数据，展示错误汇总
- 错误提示格式：`行N · 范式"XXX" · 原因：环节名称为空`

### 3.1 拖拽交互

**任务条拖拽**

**拖拽排期确认弹窗**

- 触发：时间轴环节条拖拽松手、完成工作日吸附后
- 标题：`确认排期变更`
- 描述区：显示被拖拽环节名称（加粗）+ 新的起止日期 `YYYY-MM-DD ~ YYYY-MM-DD`
- 四个按钮从左到右排列：
  - `取消`（ghost 样式）——还原原排期，不写入任何变更
  - `仅保存当前环节`（ghost 样式）——仅更新该环节排期，其余环节不变
  - `联动调整（按依赖重算）`（ghost 样式）——以变更后日期为锚点，严格按 FS 依赖链重新计算所有关联环节；**前移与后移均生效**（依赖满足即前移）
  - `联动调整（整体平移）`（primary 样式，默认焦点）——将所有有依赖关系的后置环节整体平移相同工作日 delta，保持相对间距；**前移与后移均生效**
- 弹窗出现动画：缩放 + 淡入，200ms
- 键盘：Enter 触发默认焦点按钮（`联动调整（整体平移）`），Esc 触发取消

### 8.3 组件规范（用于可维护性）

- 关键组件必须具备一致的状态覆盖：加载/空态/错误/离线/冲突
- 关键交互必须具备二次确认：删除、清空本地缓存、保存拖拽排期、字段变更
- Header必须始终展示账号与同步状态（未登录/同步中/离线待同步/冲突）
